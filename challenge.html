CS31 Final Review Challenges

If you can do these, you can probably ace the final.

Challenge 1

Dynamic Allocation, Pointers, Characters, Loops

Constraints: You may not use any libraries.

Task: Dynamically allocate an array of character pointers of length 20 (meaning the array has been allocated space for 20 elements) called:

charPointers

Then, create a for loop to assign each pointer to characters starting from ASCII code 79 and continuing backwards but skipping over every other character.

For example, if your array size is 5 and you start from ASCII code 100 or 'd', your dereferenced array would be:

{ 'd', 'b', '`', '^', '\' }

resembling the ASCII codes:

{ 100, 98, 96, 94, 92 }

Challenge 2

Vectors, Functions, Loops

Constraints: You will use the <vector> library

Task: Create a function that returns a std::vector called multiples()

This function takes in two parameters:

int multiple, int max

It will return a std::vector containing all the multiples of the multiple parameter starting from 1 until the max parameter (inclusive).

For example,

multiples(3, 30)

will return a std::vector:

{ 3, 6, 9, 12, 15, 18, 21, 24, 27, 30}

Challenge 3

Enumerations, Classes,
Constructor Overloading

Constraints: You will not use any library

Task: You will be creating one class and two scoped enumerations.

Create a scoped enumeration (enum class/enum struct) called Armor

This type will contain the possible values

Diamond Gold Iron Leather

(no Netherite sorry)


Create a scoped enumeration called Item

This type will contain the possible values

Sword Axe Stick Carrot



Create a class called MinecraftZombie

This class will have private members

double health
double damage
Armor armor
Item item

There will be a default constructor that defaults the private members to:

health = 20.0
damage = 3.0
armor = Armor::Leather
item = Item::Carrot

There will be an overloaded constructor that allows you to initialize all 4 members.



This class will have public accessors/mutators:

double getHealth()

Returns health.

double getDamage()

Returns damage.

Armor getArmor()

Returns armor.

Item getItem()

Returns item.

void setHealth(double newHealth)

Modifies health. If newHealth is less than 0, don't change health.

void setDamage(double newDamage)

Modifies damage. If newDamage is less than 0, don't change damage.

void setArmor(Armor newArmor)

Modifies armor.

void setItem(Item newItem)

Modifies item.



This class will also have these public methods:

double doDamage()

Returns damage times a multiplier based on item.

If health is less than or equal to zero, cannot do damage.

If item is Axe, return 2.5x damage

If item is Sword, return 2x damage

If item is Stick, return 1.2x damage

If item is Carrot, return 0.5x damage

void takeDamage(double damageTaken)

Reduces health by damageTaken times multiplier based on armor.

If health is less than or equal to zero, cannot take damage.

If armor is Diamond, take 0.1x damage

If armor is Gold, take 0.25 damage

If armor is Iron, take 0.5x damage

If item is Leather, take 0.9x damage
