Identifiers

Valid identifiers cannot be keywords. Can only contain letters, numbers, and underscore. Cannot start with number. Case sensitive.

int money; // Identifier for integer
float mass_table; // Identifier for float
char _letter; // Identifier for char

// Invalid Identifiers
int big money; // No spaces
float 123abc; // No leading digits
char monk*ey; // No symbols

Initialization and Assignment

Declaration: creating a new identifier without specifying its value.
Assignment: gives the identifier a value, or changes existing value.
Initialization: basically combines declaration and assignment together to give the identifier an initial value. There are several types of initialization. 

int money; // Identifier for integer
float mass_table; // Identifier for float
char _letter; // Identifier for char

// Invalid Identifiers
int big money; // No spaces
float 123abc; // No leading digits
char monk*ey; // No symbols
