How does it work?
In short: this tester iterates through all of the test arrays and compares them to the true answer arrays.

Case Coverage:
This covers all "n" values from -1 to the specified "Words per Test Array" value.
For the shiftLeft() function, it covers 8 different "amount" values: -1, 0, 1, half of "Words per Test Array" value rounded-down, half of "Words per Test Array" value rounded-up, "Words per Test Array" value, "Words per Test Array" value + 1, and 2 X "Words per Test Array" value.

Function Specs:

testArrays[]: Generates a flattened 2-dimensional dynamic array of strings. Each "row" is considered a test string array.

locateMaximumArray[]: Generates ... of correct maximum index values with dimensions (Words per Test Array + 2) x Number of Test Arrays.

countFloatingPointValuesArray[]: Generates ... of correct floating point counts with same dimensions.

hasNoCapitalsArray[]: Generates ... of correct boolean capitalization values with same dimensions.

shiftLeftArray[]: Generates a flattened 3-dimensional dynamic array of strings of dimensions ((Words per Test Array + 2) x 8 x Words per Test Array). This array records the correctly changed string arrays based on the first test array.

shiftLeftArray[]: Generates ... of correct shiftLeft outputs of dimensions (Words per Test Array + 1) x 8.

shiftLeftArray[]: Generates an array of shiftLeft test amounts.
